# Домашние задания по курсу "Основы программирования на C++"

Этот репозиторий создан для хранения решений домашних заданий по дисциплине **"Основы программирования на C++"**, преподаваемой на первом курсе образовательной программы **"Программная инженерия"** Национального исследовательского университета "Высшая школа экономики" (НИУ ВШЭ).

Автор: **Германов Артём** БПИ 248

## Структура репозитория

Каждое задание размещено в отдельной папке и содержит исходные файлы решений, а также, при необходимости, дополнительные материалы.

## Назначение

Репозиторий служит в учебных целях и демонстрирует освоение базовых концепций программирования на языке C++, включая:

- переменные и типы данных  
- управляющие конструкции  
- функции  
- массивы и строки  
- указатели  
- основы работы с файлами  
- простые алгоритмы и структуры данных
